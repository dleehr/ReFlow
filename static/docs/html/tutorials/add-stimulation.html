Tutorial - Adding a StimulationÂ¶

Note

Adding a stimulation requires the project-level permission “Add Project Data”. See User Accounts & Permissions

  1. Login to the ReFlow server.

  2. From the ReFlow home page, navigate to the project by clicking on the project link.

  3. From the project detail page, click the Add Stimulation link.

    Note

    Stimulations can also be added from the Stimulation list page by clicking on the Add Stimulation button located near the upper right of that page (beside the main heading).

  4. A new modal window will appear displaying the form fields.

  5. Fill out the form fields. Note, Name is required and a project cannot have duplicate stimulation names. The description field is optional. Once the form is completed, click the Save button.

    Note

    If there were validation errors with the submitted form, the form will be re-displayed along with the validation errors.

  6. Once the new stimulation is created, the modal window will close.
