Tutorial - Adding a SiteÂ¶

Note

Adding a site requires the project-level permission “Add Project Data”. See User Accounts & Permissions

  1. Login to the ReFlow server and navigate to the desired project by clicking on the project link.

  2. From the project detail page, click the Add Site link.

    Note

    Sites can also be added from the Site list page by clicking on the Add Site button located near the upper right of that page (beside the main heading).

  3. A new modal window will appear displaying the form fields.

  4. On the site form, enter the site name in the required Site Name field. A project cannot have duplicate site names. Once the form is completed, click the Save button.

    Note

    If there were validation errors with the submitted form, the form will be re-displayed along with the validation errors.

  5. Once the new site is created, the modal window will close.
