Tutorial - Adding a New Project
===============================

.. note:: Adding a new project requires superuser privileges.  See :doc:`../permissions`

#.  Login to the ReFlow server.

#.  On the home page, click the **Add Project** button near the upper right.

    .. image:: ../images/home-superuser.png

#.  A new project modal window will appear.

    .. image:: ../images/new-project-modal.png

#.  Fill out the form fields. Note the **Project Name** is required and ReFlow disallows duplicate project names. The description field is optional. Once the form is completed, click the **Save** button.

    .. note:: If there were validation errors with the submitted form, the form will be re-displayed along with the validation errors.

#.  Once the new project is created, the modal window will close and the project list will automatically update showing the new project.
