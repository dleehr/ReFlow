Tutorial - Adding a Site
========================

.. note:: Adding a site requires the project-level permission "Add Project Data".  See :doc:`../permissions`

#.  Login to the ReFlow server.

#.  From the ReFlow home page, navigate to the project by clicking on the project link.

#.  On the project home page, click the **Sites** link.

    .. image:: ../images/project-home.png

#.  From the **Sites** page, click on the **Add Site** button located near the upper right of the page (beside the main heading). This will take you to the site form.

    .. image:: ../images/add-site.png

#.  On the site form, fill out the fields. Note a project cannot have duplicate site names. Once the form is completed, click the **Save** button.

    .. image:: ../images/add-site-form.png

    .. note:: If there were validation errors with the submitted form, the form will be re-displayed along with the validation errors.

#.  Once your new site is created you will be re-directed to the project's site page, listing all the sites for that project.

    .. image:: ../images/subject-form-success.png
